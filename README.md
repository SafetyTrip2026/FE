# SafetyTrip2026 FE

React/TypeScript frontend for the SafetyTrip MVP.

## Core Scenario

```text
8월 초에 부모님 모시고 부산 해운대 가는데 주의할 게 있을까?
```

The screen shows:

- Region: 부산 해운대구
- Month: 8월
- Companion: 고령자
- Risks: 폭염, 호우, 태풍
- Mock answer streamed into the result panel
- Citations: `GUIDE-HEAT-ELDERLY-001`, `GUIDE-RAIN-FLOOD-002`

## Local Run

```bash
npm install
npm run dev
```

## Verification

```bash
npm run ci
```
